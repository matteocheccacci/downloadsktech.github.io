Panoramica e Funzionalità

KTS Volley Scoreboard è un'applicazione desktop progettata per fornire una soluzione completa e professionale per la gestione di un tabellone interattivo per le partite di pallavolo. Il software permette anche di gestire tabelloni pubblicitari, dirette tv e molto altro. Viene aggiornato costantemente per migliorare le funzionalità esistenti e per aggiungerne di nuove.

Visuali Disponibili

  • Tabellone: Visualizzazione a schermo intero altamente leggibile, ideale per grandi schermi. Include punteggio in tempo reale, conteggio set, sostituzioni, videocheck. Supporta l'inserimento di loghi, nomi squadre personalizzati, timer e banner sponsor.
  • Diretta: Interfaccia grafica ottimizzata per l'integrazione professionale in streaming o registrazioni (per esempio sfruttando OBS). Offre opzioni Chroma Key (verde/blu) o sfondo trasparente e supporta banner sponsor a comparsa dinamici durante il gioco e i timer.
  • Sponsor: Interfeccia grafica dedicata alla visualizzazione di banner pubblicitari, supporta anche video YouTube.
  • Pannello Admin: Interfaccia di controllo intuitiva per la gestione completa di tutte le dinamiche di gioco e configurazione, accessibile tramite web browser locale protetto da password.
  • Pannello Regia Avanzata: Interfaccia di controllo per la gestione degli sponsor, banner pubblicitari, accessibile da web browser locale protetto da password.

Funzionalità Complete

  • Gestione Punteggio e Set: Incremento/decremento rapido del punteggio, assegnazione set e logica automatica di vittoria set/match basata sulle regole configurate.
  • Controllo Servizio e Campo: Assegnazione del battitore e funzionalità di Cambio Campo durante il set (manuale) o all'intervallo set (automatico) e durante il tie-break (al punto configurato, con animazione opzionale).
  • Timeout e Sostituzioni: Tracciamento visivo e numerico dei Timeout (con limite configurabile) e delle Sostituzioni (con limite configurabile) usati da ciascuna squadra.
  • Sistema Timer Professionale: Gestione automatica e manuale di tre tipologie di timer sovrapposti:
    • Countdown Partita: Conto alla rovescia fino all'orario di inizio gara configurato.
    • Timeout Tecnico: Timer per la durata configurabile dei timeout (es. 30 secondi).
    • Intervallo Set: Timer per la durata configurabile tra i set.
  • Regole di Gioco Flessibili: Configurazione completa di tutti i parametri regolamentari: tipo di match (migliore di 3 o 5, fisso 3 o 5), punti set, punti tie-break, scarto minimo per set e tie-break, durata timeout e intervalli.
  • Personalizzazione Grafica Avanzata: Caricamento di loghi squadra, logo evento, immagine di sfondo per la vista Stadio e gestione di un carosello di immagini sponsor per la vista TV/Overlay.
  • Configurazione Viste Multiple: Possibilità di abilitare viste aggiuntive (Mirror View) su porte diverse per la connessione di schermi secondari.
  • Messaggistica e Overlay: Visualizzazione di overlay a schermo intero per il Cambio Campo automatico al Tie-Break, l'Annuncio del Vincitore e la visualizzazione dello Storico Set durante l'intervallo.
  • Clock di Sistema: Visualizzazione opzionale dell'orologio di sistema (ora corrente) sulla Vista Stadio.
  • Salvataggio delle impostazioni in Cloud: Il tuo computer si è bloccato? Non ti preoccupare, eseguendo l'accesso a un KekkoTech ID le tue impostazioni verranno sincronizzate automaticamente con il cloud.

Dettagli Tecnici

  • Architettura: Applicazione desktop basata su un framework di runtime per applicazioni web (Electron) che consente l'esecuzione nativa su sistemi operativi desktop.
  • Comunicazione Real-Time: Sfrutta un'architettura client-server locale per la sincronizzazione istantanea dei dati tra il pannello di controllo e tutte le schermate di visualizzazione, garantendo un aggiornamento immediato del punteggio.
  • Persistenza Dati: I dati di gioco e la configurazione globale sono salvati in file locali nella directory dei dati utente del sistema, assicurando che lo stato della partita e le impostazioni personalizzate vengano mantenute tra le sessioni. Inoltre viene eseguita periodicamente una sincronizzazione con il cloud associato al tuo KekkoTech ID.
  • Indirizzamento e Porta: Il software opera come un server web locale con un indirizzo IP di rete locale, permettendo l'accesso all'interfaccia di amministrazione e alle viste secondarie (TV, Stadio) da qualsiasi dispositivo connesso alla stessa rete.
  • Gestione dell'Esecuzione: Implementa un controllo per l'istanza singola per prevenire conflitti di porta e garantire la stabilità operativa.
